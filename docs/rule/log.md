### 設計方針

| ログレベル | 用途                                     |
|-----------|------------------------------------------|
| `debug`   | ローカル開発での詳細情報（本番では無効）    |
| `info`    | 正常な業務イベント（ログイン、CRUD成功等） |
| `warning` | 想定内の異常（不正な入力、存在しないリソースへのアクセス） |
| `error`   | アプリケーションエラー（AppException等）  |
| `critical`| 予期しない致命的エラー                    |
| `alert`   | 即時対応が必要                    |


### 基本的な構文

#### ファサードを使用
```php
use Illuminate\Support\Facades\Log;


// 基本的な使用方法
Log::info('ユーザーがログインしました');

// 例外をログに記録
try {
    // 何らかの処理
} catch (Exception $e) {
    Log::error('エラーが発生しました', ['exception' => $e]);
}
```
